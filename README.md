## 📊 AI 기술 설명

<div align="center">
  <img src="https://github.com/user-attachments/assets/d147dca8-d5ba-4e25-a2ea-e5de0d3cfe1f" alt="AI 아키텍처" width="80%"/>
</div>

### 멀티모달 Transformer 아키텍처

1. **입력 데이터 처리**:
   - 열화상 이미지: 16×16 패치로 분할하여 ViT에 입력
   - 센서 데이터: 온도, 전류, 미세먼지 등 수치형 데이터를 임베딩

2. **특징 추출**:
   - **ViT Transformer Encoder**: 이미지 패치의 공간적 관계 학습
   - **Soft Label Encoder**: 센서 데이터의 시간적 패턴 및 상관관계 학습

3. **정보 통합**:
   - **Cross Attention**: 이미지-센서 데이터 간 관계성 모델링
   - 두 모달리티의 정보를 통합한 고차원 특징 벡터 생성

4. **상태 분류**:
   - **Condition Classifier**: 통합된 특징 벡터를 기반으로 장비 상태 분류
   - 4단계 분류: 정상(0), 관심(1), 주의(2), 위험(3)

### 모델 성능 비교

| 모델 유형 | 사용 데이터 | 정확도 | 특징 |
|:-------:|:-------:|:------:|:------|
| **단일모달** | 이미지만 | ~53% | ViT 기반, 이미지 패턴 분석에 한정 |
| **멀티모달** | 이미지+센서 | ~90% | Cross Attention으로 데이터 통합, 복합적 패턴 감지 |

## 💰 비즈니스 가치

### 경제적 효과
- **장비 고장 비용 절감**: 500억 원 손실(30분 공정 중단) → 사전 예방으로 비용 회피
- **생산성 향상**: AI 기반 예지보전 도입 시 생산성 약 10% 향상
- **유지보수 최적화**: 설비 보전 비용 10% 절감, 매출 74% 상승

### 비즈니스 모델 최적화
- **Recall 중심 최적화**: 고장 미감지(FN) 최소화 최우선
- **비용 분석**: 
  - 고장 미감지(FN) 비용: 수십억 원 (생산 중단)
  - 불필요한 유지보수(FP) 비용: 수천만 원
- **ROI**: 정확도 90%의 AI 모델 적용 시 하루 약 80만 원 순이익
